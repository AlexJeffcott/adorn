### Developer area
#### Get started
```shell
npm i
npx lerna bootstrap
npx lerna run start --stream --scope annotate
npx lerna run start --stream --scope frontend-react
npx lerna run lint --stream --scope frontend-react
```

#### update deps
```shell
npx npm-check-updates -u
```
